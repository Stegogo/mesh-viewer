## Controls
W/S || ⇧/⇩	: Rotate mesh (z axis)
A/D || ⇦/⇨	: Rotate mesh (x axis)
Mouse Wheel	: Zoom in/out
Ctrl + N	  : New empty scene
Ctrl + O	  : Open new file
Ctrl + Q	  : Quit
